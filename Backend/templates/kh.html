1)  System Requirements

****************************************

 

MINIMUM SYSTEM REQUIREMENTS:

Windows 2000/XP

Intel Pentium III 800MHz or equivalent processor (Athlon/Duron/Celeron)

256 MB RAM

DVD-ROM Drive

NVIDIA GeForce 3 or ATI Radeon 8500 video card

DirectX 8.1 compatible sound card

DirectX 9.0c or higher

1.5GB free hard disk space

 

RECOMMENDED SYSTEM REQUIREMENTS:

Windows 2000/XP

Intel Pentium IV 1.4GHz processor

512 MB RAM

DVD-ROM Drive

NVIDIA GeForce 4 Ti, ATI Radeon 9600, or higher

DirectX 8.1 compatible sound card

DirectX 9.0c or higher

4.7GB free hard disk space

 

SUPPORTED CHIPSETS:

NVIDIA - GeForce 6800, GeForce FX (5950, 5900, 5800, 5600, 5200), GeForce4 Ti, GeForce3

ATI - Radeon x800, 9800, 9700, 9600, 9500, 9200, 9000, 8500

 

It may be necessary to update your hardware drivers before playing.

  Controls

****************************************

 

Default Keyboard Controls

 

UP                up arrow

DOWN              down arrow

LEFT              left arrow

RIGHT             right arrow

LONG PASS         D

SHORT PASS        X

THROUGH PASS      W

SHOOT             A

CURSOR CHANGE     Q

DASH              E

STRATEGY/ATT/DEF  Z

SPECIAL CONTROLS  C

PAUSE             space bar        

HELP*             F1

 

________________________________________

 

Default Game Pad Controls

 

UP                Y Axis -

DOWN              Y Axis +

LEFT              X Axis -

RIGHT             X Axis +

LONG PASS         Button 1

SHORT PASS        Button 2

THROUGH PASS      Button 4

SHOOT             Button 3

CURSOR CHANGE     Button 5

DASH              Button 6

STRATEGY/ATT/DEF  Button 7

SPECIAL CONTROLS  Button 8

PAUSE             Button 0         

HELP*             Button 9

 

*The HELP option only appears on some menus.

 

To change your button mapping for your game pad, use the “settings” program that is located in your “Pro Evolution Soccer 6 DEMO” folder.
